# External Sources Reference

## Primary News Sources

| Source | URL Pattern | Best For |
|--------|------------|----------|
| Formula1.com | formula1.com | Official results, standings, schedules, race reports |
| FIA.com | fia.com | Official stewards' decisions, regulations, technical directives |
| Autosport | autosport.com | Breaking news, technical analysis, race reports |
| Motorsport.com | motorsport.com | Global motorsport news, technical features, interviews |
| RaceFans.net | racefans.net | Independent race analysis, data-driven reporting |
| The Race | the-race.com | Expert analysis, driver/team deep dives |
| PlanetF1 | planetf1.com | News aggregation, driver market, reaction pieces |
| GPFans | gpfans.com | News, quotes, standings updates |

## Team Sources

- Official team sites (e.g., redbullracing.com, mclaren.com, ferrari.com) for breaking driver/team news
- Team social media accounts for car launch photos, livery reveals, and announcements
- Team press officers for official statements on incidents and penalties

## Calendar & Schedule

- **Official calendar**: formula1.com/en/racing/2026
- **Session times**: Always verify against formula1.com — local times vary by circuit
- **Convert all times to ET** with local time noted in parentheses

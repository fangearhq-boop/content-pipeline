# External Sources Reference

## Primary News Sources

| Source | URL Pattern | Best For |
|--------|------------|----------|
| ESPN NBA | espn.com/nba | Scores, schedules, standings, breaking news |
| NBA.com | nba.com | Official stats, injury reports, schedules |
| The Athletic | theathletic.com/nba | In-depth analysis, insider reporting |
| Bleacher Report | bleacherreport.com/nba | News aggregation, trade rumors |
| HoopsHype | hoopshype.com | Salaries, contracts, rumors |
| The Ringer | theringer.com/nba | Analysis, draft coverage, features |

## Stats Sources

| Source | URL Pattern | Best For |

| Basketball Reference | basketball-reference.com | Historical stats, player pages, game logs |
| NBA.com/stats | nba.com/stats | Official advanced stats, tracking data |

## Team Sources

- Official team sites (e.g., celtics.com, lakers.com) for breaking roster news
- Team beat reporters on X/Twitter for real-time updates
- Team PR accounts for official injury report updates

## Social & Sentiment Sources

| Source | Best For |
|--------|----------|
| r/nba (Reddit) | Fan reaction, trending stories, highlight clips |
| NBA Twitter | Breaking news, player reactions, real-time commentary |
| RealGM Forums | Trade discussion, roster analysis, salary cap breakdowns |

## Injury Reports

- **NBA Official Injury Report**: Mandatory pregame injury reports filed by all teams
- Published daily on NBA.com by 5:00 PM ET (for that night's games)
- Statuses: Out, Doubtful, Questionable, Probable, Available

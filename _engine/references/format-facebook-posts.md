## Per-Story Format

Repeat this block for each story (1 through N). Each story gets a long-form post + an image caption.

### STORY {N}: {STORY_TITLE}

#### Long-Form Post — {POSTING_WINDOW}

```
{Hook — 1-2 punchy sentences. The headline that stops the scroll.}

{Context — background, stats, what happened. 2-3 sentences.}

{Detail/analysis — the take, the why-it-matters. 2-3 sentences.}

{Closing — question, CTA, or forward-looking statement. 1-2 sentences.}
```

**Spacing rules:** Use blank lines between every paragraph. Facebook posts are read on mobile — short paragraphs with breathing room between them. No hashtags, no emojis in long-form.

#### Image Caption — {POSTING_WINDOW}

```
{1-2 sentence caption to pair with the story's image.}
```

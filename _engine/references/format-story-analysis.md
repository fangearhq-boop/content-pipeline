## Story Analysis Workflow Checklist

### For Each Story:
- [ ] Story tier matches daily brief assignment
- [ ] 3 freshness tags selected
- [ ] Summary is 1-2 sentences
- [ ] Relevance explanation is 2+ paragraphs
- [ ] 5 unique angles identified with talking points
- [ ] Best platform(s) assigned for each angle
- [ ] 2 engagement hooks developed (question + opinion)
- [ ] 3 audience sentiment perspectives documented
- [ ] 3 headline options provided (news/engagement/voice-specific)

### Overall Story Analysis:
- [ ] All 5-8 stories from research-notes included
- [ ] All angles are distinct and don't repeat across stories
- [ ] Engagement hooks align with {BRAND_NAME} audience psychology
- [ ] Headlines are platform-appropriate length
- [ ] Cross-story narrative summary identifies themes
- [ ] Ready for platform-specific content creation (X posts, Facebook posts, etc.)

## Verification Priority

When fact-checking, use sources in this order:

1. **Scores**: Official tournament site → Flashscore → ESPN
2. **Rankings**: atptour.com / wtatennis.com → live-tennis.eu
3. **Career stats**: TennisAbstract → ATP/WTA official stats
4. **Match times**: Official tournament order of play (convert to ET)
5. **Ages**: ATP/WTA player profiles (birth date) → calculate manually
6. **Draw/seedings**: Official tournament site → ATP/WTA tournament page
